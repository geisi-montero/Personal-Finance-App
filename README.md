# Finanzas Personal

Aplicación de escritorio para la gestión de finanzas personales, desarrollada con C#, .NET 8, WPF y SQLite.

---

## Características

- Registro y gestión de ingresos y gastos.
- Administración de presupuestos por categoría.
- Creación de categorías personalizadas.
- Dashboard con métricas financieras y visualización de datos.
- Almacenamiento local mediante SQLite.

---

## Tecnologías

- C#
- .NET 8
- WPF
- SQLite
- CommunityToolkit.Mvvm
- LiveCharts

---

## Arquitectura

El proyecto implementa el patrón MVVM (Model-View-ViewModel), manteniendo una separación clara entre la interfaz de usuario, la lógica de negocio y el acceso a datos.
